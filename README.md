### 🛡️ RakshaSetu — Smart Tourist Safety Monitoring System
First-author research system for real-time tourist safety monitoring. **IEEE CONIT 2026** — accepted and presented.

`LSTM Autoencoders` `Geo-Fencing` `Ethereum` `Solidity` `Full-Stack Mobile`

- LSTM autoencoder for GPS trajectory anomaly detection via sequence-to-sequence reconstruction
- 92% anomaly detection accuracy · sub-2-second alert response time
- Simulated dataset of 250,000+ GPS samples across 15,000 registered users
- Geo-fencing (ray-casting algorithms) + blockchain-based digital identity layer
